Validador de archivo de datos

Estimado usuario, en esta plataforma fue creada con la intención de validar la información proporcionada por usted. De manera tal, que los archivos proporcionados posean la información exacta que se requiere para el proyecto.

Los formatos de archivo que se aceptan son Excel (.xlsx) y Comma Separated Values (.csv)

El nombre de los archivos deben estar incluidos dentro de los siguientes.

  • mesas
  • listas
  • candidatos
  • candidaturas
  • mesas_certificados
  • mesas_candidaturas_seguridad_trep

Nota: No se aceptaran archivos que no posean alguno de estos nombres.

Cada uno de los archivos debe cumplir con las condiciones redactadas a continuación.

  • Con la excepción del archivo 'candidatos', el restante no debe poseer celdas de datos vacías.
  • La cantidad de columnas dentro del archivo debe ser la requerida.
  • El nombre de las columnas debe estar dentro de los nombres esperados, de manera ordenada o desordenada.
  • La informacion dentro de la celda debe pertenecer al tipo de dato que se espera.
